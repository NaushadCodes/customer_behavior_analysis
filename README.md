# 👨🏻‍💻 Customer Behavior Data Analysis Project

This project demonstrates an **end-to-end data analytics workflow** using customer shopping behavior data.  
The objective is to analyze raw data and transform it into meaningful insights that can support business decision-making.

---

## 🎯 Project Objective

- Analyze customer purchasing behavior  
- Identify trends and patterns  
- Segment customers based on their activity  
- Generate insights for better business decisions  

---

## 🧰 Tools & Technologies Used

- **Python (Pandas, NumPy)** → Data Cleaning & EDA  
- **SQL (PostgreSQL/MySQL)** → Data Analysis & Querying  
- **Power BI** → Dashboard & Visualization  

---

## 📊 Project Workflow

### 1️⃣ Data Cleaning & Exploration (Python)
- Imported dataset using Pandas  
- Handled missing values and duplicates  
- Performed exploratory data analysis (EDA)  
- Identified initial trends and patterns  

### 2️⃣ Data Analysis (SQL)
- Loaded cleaned data into a SQL database  
- Wrote queries to answer business questions  
- Analyzed:
  - Customer segments  
  - Purchase behavior  
  - Revenue trends  

### 3️⃣ Data Visualization (Power BI)
- Built an interactive dashboard  
- Visualized KPIs and important metrics  
- Created charts for trends and comparisons  

### 4️⃣ Insights & Reporting
- Summarized findings  
- Generated insights  
- Provided basic recommendations  

---

## 📁 Project Files

- `Customer_Shopping_Behavior_Dataset.ipynb` → Raw Dataset 
- `Customer_Shopping_Behavior_Analysis.ipynb` → Python analysis  
- `customer_behavior_sql_queries.sql` → SQL queries  
- `customer_behavior_dashboard.pbix` → Power BI dashboard
